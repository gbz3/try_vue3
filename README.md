# 

## 初期化

```bash
$ npm init vite-app try_vue3
$ cd try_vue3
$ npm install
$ npm run dev
```

URL: `http://localhost:3000/`

## GitHub 登録

```bash
$ git init
$ git add .
$ git commit -m "initial commit"
```

- GitHub に空リポジトリ作成

```bash
$ git branch -M main
$ git remote add origin git@github.com:gbz3/try_vue3.git
$ git push -u origin main
```
